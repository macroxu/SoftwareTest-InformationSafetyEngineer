# 2  编辑网络配置

## 2.1 修改网卡配置信息
```shell
cd /etc/sysconfig/network-scripts 

vim  ifcfg-ens160 #修改本地网卡配置文件,linux 版本的不同，具体的细节也不一致
TYPE=Ethernet
PROXY_METHOD=none
BROWSER_ONLY=no
BOOTPROTO=none
DEFROUTE=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=no
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_FAILURE_FATAL=no
IPV6_ADDR_GEN_MODE=stable-privacy
NAME=ens160
UUID=1111-2aff-40ee-b912-111
DEVICE=ens160
ONBOOT=yes
IPADDR=10.0.0.63
PREFIX=24
GATEWAY=10.0.0.3
DNS1=10.0.0.50
DNS2=114.114.114.114

```

## 2.2 修改DNS 和本地host

> 修改DNS
```shell
cd /etc/sysconfig/network-scripts  #在这个目录下，找到对应的配置文件修改DNS1 DNS2

vim ifcfg-ens160

修改 DNS1 DNS2 的值

```

```shell
cd /etc/

vim host
```
